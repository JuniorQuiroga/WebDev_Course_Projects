Hello World

Like the tradition demands we have to start our jorney with the now classic hello world aplicaction.

To make our hello world app we have to create a new python file, lets call it "app.py", inside this we have to define a window and a label inside with the text "Hello world!".

import sys
from PyQt5.QtWidgets import QApplication, QLabel

class MainWindow(QMainWindow):
 def __init__(self):
  super().__init__()
  self.setWindowTittle("Hello world")
  self.label = QLabel("Hello world!")
  self.setCentralWidget(self.label)

def main():
 app = QApplication(sys.argv)

 window = MainWindow()
 window.show()

 app.exec()

if __name__=="__main__":
 main()

What are we doing?

  • from PyQt5.QtWidgets import QApplication, QLabel we import the classes that we are going to use.
  • app = QApplication(sys.argv) this creates an instance of QApplication.
  • window = MainWindow() to create a custom window we create a subclass of QMainWindow.
  • in the __init__() we configure our custom window.
  • self.setWindowTittle("") we stablish the title.
  • self.label = Qlabel("") we create a new label and with self.setCentralWidget(self.label) we assign it to the custom window.
  • window.show() is a necesary step in order to see our custom window.
  • we call app.exec() to start the event loop.
Window sizing

We can stablish the size of our window with the methods

.setFixedSize(QSize(x_size,y_size))

on our __init__() method in the custom window subclass. We can also stablish restrictions such as maximum and minimum size with the methods:

.setMinimumSize(QSize(x_size,y_size)) .setMaximumSize(QSize(x_size,y_size))
Buttons

Buttons are a fundamental part of any user interface to interact with the software.

We can create buttons easily in PyQt5 with the next peace of code inside our custom window "__init__()", but first we have to import the buttons module called QPushButton

from PyQt5.QtWidgets import QPushButton

Now we can create our button, in this case we're going to create a simple button that will print something when we push it

class MainWindow(QMainWindow):
 def __init__(self):
  super().__init__()
  self.button = QPushButton("Press me")
  self.button.clicked.connect(self.the_button_was_clicked)

  self.setCentralWidget(self.button)

 def the_button_was_clicked(self):
  print("Hello!! :D")

with this new implementation of our custom sub class of MainWindowe we should see a button in the middle with the text Press me and when we press it, it should show us a Hello!! :D on the console.

So what's happening here?

  • self.button = QPushButton("Press me") first we create a new button with a text to show in the middle.
  • self.button.clicked.connect(self.the_button_was_clicked) then we state that when we click the button it will call the the_button_was_clicked() method inside of our custom subclass
  • the_button_was_clicked() once this method is called it will run what ever we specified inside, in this case it will print Hello!! :D
Text input

Other important way to interact with a piece of software is via input fields the most commondly known being the text input, a way to input data to be procesed.

We can make this easily in PyQt5, but before anything we have to import the corresponding module, in PyQt5 this module is called QLineEdit.

from PyQt5.QtWidgets import QLineEdit

But for this case this is not going to be enough because we need to pack two elements in the window, the text field and a label, for this we need to import the QWidget, the QLabel and the QVBoxLayout module.

from PyQt5.QtWidgets import QWidget, QLabel, QVBoxLayout

Now we can instanciate a new text field and a label that will show whatever we write on the field.

class MainWindow(QMainWindow):
 def __init__(self):
  super().__init__()

  self.input = QLineEdit()
  self.label = QLabel()

  self.input.textChanged.connect(self.label.setText)

  layout = QVBoxLayout()
  layout.addWidget(self.input)
  layout.addWidget(self.label)

  container = QWidget()
  container.setLayout(layout)

  self.setCentralWidget(container)

So what's happening here?

  • self.input = QLineEdit() we instanciated a new text fiel input follow by instanciation of the label.
  • self.input.textChanged.connect(self.label.setText) we declare that if the text changes it sends a signal to the setText method of the label that repleces the current text of the label for the one introduced in the field.
  • layout = QVBoxLayout() we instanciate a new layout.
  • layout.addWidget(self.input) layout.addWidget(self.label) we add the field and label to the layout
  • container = QWidget() we declare a new widget that will contain everithing, this is going to ve the central component of our custom MainWindow.
  • container.setLayout(layout) we set our layout to our container and this is the way we can pack multiple elements in one window.
Input processing

To coclude this begginer guide let's create a simple app that will take text from a field and uppercase it with the press of a button.

from PyQt5.QtWidgets import QLineEdit

But for this case this is not going to be enough because we need to pack two elements in the window, the text field and a label, for this we need to import the QWidget, the QLabel and the QVBoxLayout module.

from PyQt5.QtWidgets import QApplication, QMainWindow, QWidget
from PyQt5.QtWidgets import QLabel, QLineEdit, QVBoxLayout, QPushButton
import sys
class MainWindow(QMainWindow):
 def __init__(self):
  super().__init__()

  self.setWindowTitle("Upper text app")

  container = QWidget()
  layout = QVBoxLayout()
  self.text_input = QLineEdit()
  self.button = QPushButton("Upper the text")
  self.label = QLabel()

  self.button.clicked.connect(self.read_and_upper_field)

  layout.addWidget(self.text_input)
  layout.addWidget(self.button)
  layout.addWidget(self.label)

  container.setLayout(layout)
  self.setCentralWidget(container)

 def read_and_upper_field(self):
  upper_text = str.upper(self.text_input.text())
  self.label.setText(upper_text)

def main():
 app = QApplication(sys.argv)

 window = MainWindow()
 window.show()

 app.exec()

if __name__=="__main__":
 main()

In our custom MainWindow subclass we declare five elements a widget that will contain the layout which in turn will contain the text field, button and label. After this, we declare that when the button is clicked, it will call the method read_and_upper_field(self), in this methos we make use of the .upper() method from the str class which receives as parameter the text from the text field with self.text_input.text(), the result of this is passed as parameter to the .setText() method of our label.
